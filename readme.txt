Для запуска бота необходимо:
1. Клонировать проект с GitHub в локальную директорию по ссылке
https://github.com/fedaligo/telegram-bot.git
2. Скачать приложение NGROK по ссылке https://ngrok.com и выполнить рекомендации по его
настройке
3. Запустить приложение NGROK и ввести в нём команду "ngrok http 5000"
4. После создания соединения скопировать полученыый адрес из приложения NGROK
(например: https://2e00ec5f79f8.ngrok.io) и вставить этот адрес в application.properties
в графу telegrambot.webHookPath
5. В браузере перейти по ссылке https://api.telegram.org/bot<token>/setWebhook?url=https://2e00ec5f79f8.ngrok.io
для создания соединения между локальным веб-сервером и сервером telegram. Где https://2e00ec5f79f8.ngrok.io - это
адресс полученный в приложении NGROK, а <token> - это токен телеграмм бота.
6. Загрузить базу из проекта на ПК. Для того чтобы flyway загрузил базу из проекта нужно,
чтобы на ПК была создана БД на базе PostgreSQL и чтобы все её реагистрационные данные были
внесены в application.properties как на примере ниже:
spring.datasource.driver-class-name=org.postgresql.Driver
  spring.datasource.url= jdbc:postgresql://localhost:5432/city_telegram_bot
  spring.datasource.username=test
  spring.datasource.password=root
6. Запустить проект.
7. Запустить приложение telegram и найти бота с именем MyFirstTelegramBot и именем пользователя
@FedaligoBot.
8. Бот реагирует на 3 ключевых слова и названия городов имеющихся в базе.
Ключевые слова:
/start - бот запросить пользователя ввести название города о котором он хочет получить информацию
help - бот предоставит краткую информацию о своем назначении
full list - бот выведет список стран и городов имеющихся в базе (количество выводимых сведений
ограничено максимальным количеством символов для одного сообщения)
moscow - бот выведет информацию о городе Москва
Если будет введено не корректное ключевое слово, то бот тоже отреагирует и при этом попросит
ввести корректное клювое слово.
9. Для управления данным в базе используется графический интерфейс swagger.
После запуска проекта доступ к нему осуществляется по ссылке http://localhost:5000/swagger-ui.html#/city-info-controller
